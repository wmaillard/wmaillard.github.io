Using CameraView to Take a Photo

Now that we have our navigation and gallery set up, we can expand our reach a little and use an external library called React-Native-Camera.  You can get set up with installation through their guide.  

Once you have React-Native-Camera installed, you're ready to use your Android's camera to take pictures!  The easiest way to start using React-Native-Camera, is to create a new class and include the <Camera> element within its rendered view.  Here is how I set mine up:

   <View style ={{flex: 1}}>
      <View style ={{flex: 1}}>
            <Toolbar title={'Upload a Photo'} icon={'arrow-back'} onIconPress={this.props.onBack} />
      </View>
  
      <Camera
              style = {{flex: 10}}
        ref={(cam) => {
          this.camera = cam;
        }}
        aspect={Camera.constants.Aspect.fill}
        captureTarget ={Camera.constants.CaptureTarget.temp}
        captureQuality = {Camera.constants.CaptureQuality.low}>
        <View style={{flex:5}}/>
        <Button style={{flex:1}} text='Take Picture and Upload' value = "NORMAL RAISED" raised={true} onPress={this.takePicture.bind(this)}>
        </Button>
      </Camera>
      </View>

As you can see, I kept the normal Toolbar at the top, for consistency.  Below that you can see the Camera element which has the ref function, which just sets the scope of the camera.  Next is the aspect which sets the aspect ratio that the camera will use (fill just uses the camera's native aspect ratio and stretches our View to fit that.  CaptureTarget sets where the image will be stored (temp allows us to store the image in the app's cache so that it both easily accessible and ephemeral).  CaptureQuality lets us set the quality of the image being taken (for testing and because these images will eventually be uploaded to a server, I have set this to low).

Below the <Camera> element there is a <Button> element which is imported from 'react-native-material-design'.  As you would expect, the onPress attribute is set to bind the takePicture function to this scope.  This just makes the button fire our takePicture function.

//Here data.path is the location of the image.  In the 'Uploading Images', section I use this path to POST the image to a web server.  This POSTing can be done here or the path can just be saved for later use
