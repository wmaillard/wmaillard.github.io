React Native Material Design

React Native Material Design is a fantastic add-on to React Native that allows you to quickly and easily make your app look pretty (so you can get some actual work done!).  The developers of this package have created somewhat complete documentation that can be used to get started.  Unfortunately, the documentation for what I think is the most useful part of this package, the "Toolbar" (or Navbar), is completely missing.  To get started using React Native Material Design, follow this tutorial.  Once you are all set, you can add buttons, cards, etc based on their documentation.  A navigation bar can be added fairly easily with the following code:

//Don't forget to import the toolbar
import { Button, Card, Toolbar } from 'react-native-material-design';
….
//At the top of a view, add the toolbar
<Toolbar title={'Gallery'} icon={'arrow-back'} onIconPress={this.props.onBack}

Where 'Gallery' is the title, 'arrow-back' is one of the 3000 icons included during the installation of the React Native Material Design, and 'this.props.onBack' is function that just does navigation.pop() as shown in this part of the React Native set up tutorial.  The combination of the <Navigator> element and the <Toolbar> element make seamlessly navigating through a React Native app easy.
